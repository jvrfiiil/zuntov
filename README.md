# Linkin Park — Учебный одностраничный фан-сайт

**Демонстрационный проект в одном файле**  
Создан для изучения и отработки HTML + CSS + JavaScript (простая навигация по секциям)

## О проекте

Один HTML-файл, который содержит:

- красивую тёмную тему
- навигацию в виде вкладок (без перезагрузки страницы)
- 8 основных разделов сайта
- адаптивный контейнер
- плавное переключение секций
- современный (но простой) внешний вид

Проект сделан в учебных целях, поэтому максимально использует базовые возможности HTML/CSS/JS без внешних фреймворков и библиотек.

## Как открыть

1. Скопируйте весь код из файла `linkinpark_all_in_one.html`
2. Создайте новый файл с этим же именем
3. Вставьте код
4. Сохраните
5. Откройте файл в любом современном браузере

Готово! Всё работает локально без сервера.

## Структура сайта (разделы)

- **Главная** — приветствие и большая картинка
- **Общая информация** — текущий и бывший состав
- **История** — кратко по периодам + якоря
- **Дискография** — список студийных альбомов
- **Фильмография** — таблица с саундтреками к фильмам
- **Источники** — полезные внешние ссылки
- **Заказать** — имитация формы заказа мерча
- **Контакты** — демонстрационные способы связи

## Технические особенности

- Переключение секций через JavaScript (без перезагрузки страницы)
- Встроенные стили в `<style>` (без внешних .css файлов)
- Простая анимация появления секции
- Тёмная тема с акцентом на фирменный синий цвет Linkin Park (#1E90FF)
- Изображение на главной — взято из Unsplash (публичное)

## Требования к браузеру

Любой современный браузер 2020+ года  
(Chrome, Firefox, Edge, Safari, Opera и т.д.)
